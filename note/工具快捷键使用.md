# 工具快捷键使用

  主要介绍一些工具使用的快捷键内容

## Typora

可以拥有不同的主题 推荐 Vue主题 和Bear 主题
- CTRL+SHIFT+L 跳出侧边栏的文件管理大纲
- Ctrl+/打开关闭源代码模式
- 支持html语法的不能实时渲染
- CTRL+SHIFT+I插入图片
- CTRL+K插入地址

#### 建仓库

两种⽅方式： 

1. ⽤用 Git 之前已经有项⽬目代码
$ cd 项⽬目代码所在的⽂文件夹 $ git init 
2. ⽤用 Git 之前还没有项⽬目代码
$ cd 某个⽂文件夹 $ git init your_project    #会在当前路路径下创建和项⽬目名称同名的⽂文件夹 $ $cd  your_project
