#### blob、tree、commit对象关系
一个commit 是当时的一个快照 里面包含了tree ，一个目录一个tree，blob是具体内容
当我们add时候就已经把内容放到.git/objects 里面中
git cat-file -p 083e18d
#### 常用命令

git add  增加 
   -u 已经被git管控的文件进行更新
git commit --m 暂存区提交到版本历史
   -am 工作区直接提交到版本历史
git reset --hard 把暂存区和工作区还原回去 
git mv 变化工作目录内容和添加到暂存区
git log 看版本演变历史
  --oneline 行看
  -n4 看最近几次提交 
  --all 查看所有分支演进历史
  --graph 图形分支查看
  temp 查看 temp 分支
git branch  查看分支
   -v 看本地多少个分支
git checkout -b 目录 ‘hash值’ 创建分支
gitk 图形界面查看版本历史
